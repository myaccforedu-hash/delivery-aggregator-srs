# SRS – Система агрегатора служб доставки

## 1. Введение
Данный документ описывает требования к системе агрегатора служб доставки.

## 2. Заинтересованные стороны
- Клиенты
- Службы доставки
- Администраторы системы

## 3. Функциональные требования
- FR1 – Регистрация пользователя
- FR2 – Создание заказа
- FR3 – Агрегация служб доставки

## 4. Нефункциональные требования
- Производительность
- Безопасность
- Доступность

## 5. Архитектура системы
- API-шлюз
- Сервис-агрегатор
- API поставщиков доставки
- База данных

## 6. Пример использования
Пользователь создаёт заказ → система выбирает оптимального поставщика доставки → заказ обрабатывается.
